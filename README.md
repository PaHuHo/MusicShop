## With Backend:
- **cd backend**
- **composer install**
- **cp .env.example .env**
- **php artisan key:generate**
- **php artisan serve (Local)**

## With Frontend:
- **cd frontend**
- **npm install**
- **npm run dev**

## With Server run
- **Change URL API 127.0.0.1**

## With Server run
- **Change URL API ip-server**
